# ElevateLP Web

App para adaptar templates Elementor com IA.

## Deploy no Railway

1. Crie conta em railway.app
2. Clique em "New Project" > "Deploy from GitHub"
3. Conecte este repositório
4. Adicione a variável de ambiente: ANTHROPIC_API_KEY=sua-chave-aqui
5. Pronto! O app fica disponível em um link público

## Variáveis de ambiente necessárias

- ANTHROPIC_API_KEY: Sua chave da API Anthropic (console.anthropic.com)
